# devnotes — A Tech Blog Landing Page

A clean, modern **tech/coding blog landing page** built with pure HTML and CSS. Designed with a terminal/code-editor aesthetic — dark theme, monospace fonts, and a developer-first visual style.

## Live Demo

🌐 [View Live Site](https://jemimoses.github.io/tech-blog/)

## Features

- **Dark theme** — GitHub-inspired dark color palette (`#0d1117` background)
- **Terminal aesthetic** — JetBrains Mono font, blinking cursor effect, file-tab styled blog cards
- **Blog post cards** — styled like code snippet files with `.md` extension tabs
- **Newsletter section** — designed as a fake terminal window (`subscribe.sh`)
- **Sticky navbar** — with blur backdrop effect
- **Fully responsive** — mobile (1 column), tablet (2 columns), desktop (3 columns)
- **Pure HTML + CSS** — no JavaScript frameworks, no dependencies

## Tech Stack

- **HTML5**
- **CSS3** (Flexbox, Grid, CSS Variables, Media Queries, CSS Animations)
- **Google Fonts** — JetBrains Mono + Inter
